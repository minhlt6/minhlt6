### 💡 Dự án Tiêu biểu

#### 📚 Academic Regulations RAG Chatbot (M_chatbot)
* **Công nghệ:** FastAPI, Qdrant, Supabase, PostgreSQL, Cross-Encoder, LLMs.[cite: 4]
* **Mô tả:** Xây dựng hệ thống Question-Answering truy xuất quy chế đào tạo với khả năng tự động lọc theo năm học.[cite: 4]
* **Kỹ thuật nổi bật:** 
  * Áp dụng **Hybrid Retrieval** (Vector Search + BM25) kết hợp Reciprocal Rank Fusion và **Cross-Encoder reranker** để tối đa hóa độ chính xác.[cite: 4]
  * Thiết kế luồng đồng bộ hóa dữ liệu tự động giữa Supabase và **Qdrant** vector database.[cite: 4]
  * Hỗ trợ real-time streaming (SSE) và quản lý lịch sử ngữ cảnh bằng asyncpg trên PostgreSQL.[cite: 4]

#### 👁️ Content-Based Image Retrieval with Deep Metric Learning
* **Công nghệ:** PyTorch, ResNet50, Graphormer, Deep Metric Learning.[cite: 3]
* **Mô tả:** Đề xuất và phát triển kiến trúc lai "ResNet-GT" để giải quyết vấn đề khoảng cách ngữ nghĩa (semantic gap) trong phân loại ảnh mịn.[cite: 3]
* **Kỹ thuật nổi bật:**
  * Nhúng khoảng cách đường đi ngắn nhất trên đồ thị K-NN vào cơ chế Multi-Head Self-Attention.[cite: 3]
  * Đạt hiệu năng **Zero-shot retrieval** cực cao (Recall@1 = 84.22%) trên tập CUB 200-2011, vượt trội hoàn toàn so với các mô hình ViT tiêu chuẩn.[cite: 3]

#### 👤 Facial Expression Recognition Web Application
* **Công nghệ:** Python, Scikit-learn, OpenCV, Streamlit.
* **Mô tả:** Xây dựng pipeline phân loại cảm xúc khuôn mặt đầu cuối.
* **Kỹ thuật nổi bật:** Tự triển khai các thuật toán trích xuất đặc trưng **HOG (Histogram of Oriented Gradients)** kết hợp với các mô hình Machine Learning truyền thống (Random Forest, KNN). Tối ưu hóa siêu tham số bằng GridSearchCV và đưa lên web app có khả năng crop khuôn mặt real-time qua webcam.

#### 📈 Financial Data Mining & Time-Series Prediction
* **Công nghệ:** Python, Pandas, Linear Regression.
* **Mô tả:** Phát triển công cụ cào dữ liệu tài chính/giá vàng lịch sử. Thực hiện kỹ thuật Feature Engineering (rolling averages, lag indicators) và ứng dụng các thuật toán hồi quy tuyến tính để dự báo xu hướng biến động giá.
